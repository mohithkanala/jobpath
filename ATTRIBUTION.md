# Attribution

jobpath builds on prior MIT-licensed open-source work in the AI job-search automation space. The original scaffolding, including the modes pattern, pipeline scripts, scoring rubric structure, and dashboard, derives from that prior art.

The methodology layer in this repository -- the resume rigor rules in `METHODOLOGY.md`, the per-job folder convention, the canonical state machine, the chat-compaction-survival CONTEXT.md pattern, the cross-agent slash command surface, and the operational discipline encoded in the modes -- is the work of the current maintainer.

Original copyright in `LICENSE` is preserved per MIT terms. Anyone redistributing jobpath downstream should preserve `LICENSE` in turn.

If you fork jobpath, you inherit both lineages. Keep `LICENSE`, keep this file, write your own opinions on top.
